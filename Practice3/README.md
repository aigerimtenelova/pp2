# Practice Python Projects – Practice-03

This repository contains Python programs that focus on **functions, lambda expressions, classes, and inheritance**.  
The project is designed for **learning, practicing, and understanding core Python concepts**.

---

## Repository Structure

- **functions** – Python functions and their features:
  - `basic_functions.py` – defining and calling functions  
  - `function_arguments.py` – positional, keyword, and default arguments  
  - `return_values.py` – returning values from functions  
  - `args_kwargs.py` – using `*args` and `**kwargs`  

- **lambda** – working with lambda expressions:
  - `lambda_basics.py` – basic lambda functions  
  - `lambda_with_map.py` – using `lambda` with `map()`  
  - `lambda_with_filter.py` – using `lambda` with `filter()`  
  - `lambda_with_sorted.py` – using `lambda` with `sorted()`  

- **classes** – object-oriented programming in Python:
  - `class_definition.py` – defining classes and creating objects  
  - `init_method.py` – using the `__init__` constructor  
  - `class_methods.py` – instance and class methods  
  - `class_variables.py` – working with class and instance variables  

- **inheritance** – inheritance concepts in Python:
  - `inheritance_basics.py` – single inheritance examples  
  - `super_function.py` – using the `super()` function  
  - `method_overriding.py` – overriding methods in subclasses  
  - `multiple_inheritance.py` – multiple inheritance examples  

---
